## Scaffolding & Basics

Requires [Node 10+](https://nodejs.org/en/download/releases/) and Angular

```
npm i -g @angular/cli
```

Create proj:

```
ng new ngBuild
ng s -o
```

Manual Build:

```
npm i
ng build --prod
```
